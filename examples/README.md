# parsec/examples

The examples in this directory will guide you on how to use different flags by
type and how to implement them in you projects.

Each directory follows and `xy-name` notation that classifies its examples:
- `x`: category of the example:
  - `0`: teaches you the basics of `flag.h`.
  - `1`: puts theory to the test with small, real programs.
- `y`: the highest, the harder. But rest assured, examples contain the least
amount of code and useful comments so they are easy to read and follow.

You can build tests with `Make` or `nob.h` (not included).

Happy coding!
