# parsec

Inspired by tsoding's [flag.h](github.com/tsoding/flag.h), which was inspired by Go's [flag](https://pkg.go.dev/flag) module.

> [!WARNING]
> Use at your own risk! This library is updated often and may not be stable.

A simple argument parser in C.

## Installation

```sh
wget https://raw.githubusercontent.com/kerudev/parsec/refs/heads/master/parsec.h
```

## How to use

You can find simple examples in the [`examples`](./examples) directory.
